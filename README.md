# SMOTE Library and Metrics Toolkit

This repository provides implementations of SMOTE-based oversampling methods and a data-level evaluation framework for imbalanced learning.

## Overview

This project accompanies the paper:

**"Clustering-Guided Oversampling and Geometric Validation for Imbalanced Learning"**

It includes:

- A library of SMOTE-based synthetic data generation methods
- A toolkit for evaluating synthetic data quality using both statistical and geometric metrics
